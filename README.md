# ln-api-boilerplate
a boilerplate project for creating lightning-enabled backend applications

## Quickstart
```sh
$ yarn
$ cp .env.example .env
$ yarn start
```

## Compile and run
```sh
$ yarn build
$ node build/index
```

Check the server is running:
```sh
$ curl -GET localhost:3000/api
    {"ok":true}
```
